Webulous server

Welcome to the EMBL-EBI/NCBO Webulous server.

Webulous Templates

Use Webulous to create spreadsheets with ontology based data validations. Webulous can process populated templates and transform them into OWL/RDF using design patterns expressed in OPPL.

Google spreadhseets

Connect to a Webulous server from Google Spreadsheets with the Webulous Google Add-On. This add on support loading templates from a Webulous server, and allows users to submit populated spreadsheets back to the Webulous server for transformation into OWL.

Set up your own Webulous

Install your own Webulous server for managing ontology building templates and connect to the server directly from Google Spreadsheets using the Webulous Add-On.
