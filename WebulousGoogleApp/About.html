The Webulous Add-on provides access to the Ontologies in the NCBO BioPortal repository so that you can easily create ontology based data validations in Google Sheets. From the Webulous menu select the "Ontology Search..." option to open a sidebar where you can access BioPortal. You can choose to search across all ontologies in BioPortal or restrict this to a specific ontology by choosing one from the dropdown.

Creating a data validation

You can create data validations in your spreadsheet directly from search results. To create a data validation first select a range of cells in your spreadsheet where you would like the validation to apply. This can be a custom range of cells or an entire column or row.

Note: Creating long lists of data validations can effect performance of the Google Sheet so it is recommended to only create validations for a few hundred cells at a time.

With a range selected, hover over the term in your search result that will act as the source for the data validation. Two buttons will appear that will allow you to create a data validation based on either direct subclasses of the term, or all subclasses (descendants) of the term. The data validation will also include the selected term. You can merge multiple ontology terms into an existing data validation. Simply select an existing range and choose either subclasses or descendnats of the new term from the search results.

Note: Data validations are currently restricted to a maximum of 5000 terms.

Removing data validation

You can remove existing data validations using the normal Data > Validations... menu item in Google Sheets. However, validations created with Webulous create hidden sheets that store the term label to URI mappings. In order to remove these validations entirely it is recommended to use the Webulous > "Ontology Validations..." menu item. This will show all the existing validations. From here you can view or remove individual validations or you can simply remove all validations to reset the spreadsheet.

Webulous templates

Webulous can also be used to edit ontology building templates from a Webulous server. For instance, you can use Webulous to submit new terms to ontologies like EFO. You can get more information about creating your own Webulous server here and the code is also available from GitHub.

You can only load one Webulous template at a time. Select the "Webulous templates..." option in the menu. Choose a Webulous template server and a list of available templates will appear. Choose the template and a new sheet will be created in your spreadsheet. Populate the columns with the required information and submit the data back, again using the "Webulous template..." menu item.
